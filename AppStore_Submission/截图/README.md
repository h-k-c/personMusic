# 📱 App Store 截图要求 — 个播

## 必需：iPhone 6.7 寸

| 项目 | 规格 |
|------|------|
| 尺寸 | 1290 × 2796 px |
| 数量 | 至少 3 张，推荐 5-6 张 |
| 格式 | PNG 或 JPG（推荐 PNG） |
| 色彩 | sRGB |

## 可选：iPad 12.9 寸（第六代）

| 项目 | 规格 |
|------|------|
| 尺寸 | 2048 × 2732 px |
| 数量 | 至少 3 张 |
| 格式 | PNG 或 JPG |

> ⚠️ 你的 App 设置了 `TARGETED_DEVICE_FAMILY = 1,2`（iPhone + iPad），
> 如果 iPad 上体验不好，建议在 pbxproj 中改为 `TARGETED_DEVICE_FAMILY = 1`（仅 iPhone），
> 这样就不需要 iPad 截图。

## 截图内容建议

因为是本地音频播放器，建议截取以下场景：

1. **空状态引导页** — 展示功能介绍 + 导入按钮
2. **播放界面** — 正在播放音频，展示进度条和播放控制
3. **文件列表** — 本地音乐列表，展示文件夹分组
4. **变速播放** — 展示 6 档变速菜单
5. **锁屏播放** — 展示锁屏控制界面（如果有 Apple Watch 更好）

## 制作方法

- **模拟器截图**：`Cmd + S` 截屏（或 `xcrun simctl io booted screenshot`）
- **真机截图**：电源键 + 音量上键
- **套壳工具**：App Store Connect 支持自动套壳，直接上传无壳截图即可
