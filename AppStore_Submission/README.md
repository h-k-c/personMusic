# 🚀 App Store 上架资料 — 个播

> 本文件夹包含「个播」上架 App Store 所需的所有资料。按下面的导航逐个完成即可。

## 📂 文件夹结构

```
AppStore_Submission/
├── README.md           ← 你在这里
├── CHECKLIST.md        ← 📋 核心：逐项检查清单，跟着做
├── 分析报告.md          ← 项目扫描结果
├── 截图/               ← 📱 截图要求 + 存放截图的目录
├── 文案/               ← ✏️ App描述、关键词、推广文本
├── 法律/               ← 🔒 隐私政策、技术支持网页
└── 配置记录.md         ← 📝 App Store Connect 配置备忘
```

## ⚡ 快速开始

1. 打开 `CHECKLIST.md` — 这是你的上架进度看板
2. 按顺序逐项完成，每完成一项更新状态
3. 文案在 `文案/` 文件夹编辑
4. 截图放入 `截图/` 文件夹
5. 法律页面在 `法律/` 文件夹，需部署到公网
