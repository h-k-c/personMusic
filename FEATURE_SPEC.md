# 个播 — 功能说明文档

> 给 Stitch 设计师：这是一份完整的功能规格，用于重新设计 UI/UX。

---

## App 定位

**个人本地音乐播放器** — 导入 iPhone 上的音频文件进来听，断网也能用。没有在线功能。

## 核心功能清单

### 1. 底部 Tab 导航（2 个）

| Tab | 名称 | 功能 |
|-----|------|------|
| 播放器 | `play.circle.fill` | 当前播放界面 + 播放控制 |
| 本地音乐 | `folder.fill` | 音乐文件浏览 + 导入 |

### 2. 播放器页面

#### 2a. 空状态引导页（无歌曲时）
- 一个大圆形 + 耳机图标
- "个播 — 你的随身音频播放器"
- 四行功能卖点（图标+文字）：
  - 导入本地音频文件或文件夹
  - 6 档变速播放
  - 自动记忆播放位置
  - 锁屏控制 + 后台播放
- "导入音频"按钮 → 跳到本地音乐 Tab

#### 2b. 播放中界面（有歌曲时）
- 专辑封面区域（正方形，圆角）
- 歌曲标题 + 艺术家
- 可拖拽进度条（显示当前时间 / 总时长）
- 播放控制按钮：
  - **上排**：上一首 | 播放/暂停（大圆按钮）| 下一首
  - **下排**：循环模式（无/列表/单曲）| 倍速（0.75x-2.0x）| 随机播放
- "播放列表"按钮 → 弹出列表 Sheet

### 3. 本地音乐页面
- 导航栏标题 "本地音乐"
- `+` 按钮 → Action Sheet 选择"添加文件夹"或"添加文件"
- 垃圾桶按钮 → 清空所有音乐（有确认弹窗）
- 列表按文件夹分 Section
- 每行显示：格式后缀（小方块）+ 文件名 + 大小·时长 + 信息按钮(i)
- 点击行 → 开始播放 + 自动跳回播放器 Tab
- 信息按钮 → 弹出详情 Sheet：文件名、标题、格式、大小、时长、文件夹路径

### 4. 播放列表弹窗
- 半屏 Sheet
- 标题 "播放列表" + 清空按钮 + 关闭按钮
- ScrollView 列表
- 当前播放的歌曲有高亮图标
- 点击切换歌曲
- 清空需二次确认

### 5. 播放控制功能

| 功能 | 说明 |
|------|------|
| 播放/暂停 | 锁屏 + 控制中心也可控制 |
| 上一首/下一首 | 支持循环模式 |
| 循环模式 | 无循环 → 列表循环 → 单曲循环 |
| 倍速播放 | 0.75x / 1.0x / 1.25x / 1.5x / 1.75x / 2.0x（popover 选择） |
| 随机播放 | 打乱播放列表顺序 |
| 进度拖拽 | 手指拖动进度条任意位置跳转 |
| 音量 | iOS 系统音量 |
| 自动记忆 | 退出 App 自动记住播放位置，下次打开恢复 |

### 6. 系统能力
- App 切到后台继续播放
- 锁屏显示歌曲信息 + 播放控制
- 来电/闹钟自动暂停，结束后可选恢复

## 数据模型

| 模型 | 关键字段 |
|------|---------|
| Song | title, artist, duration(秒), url |
| MusicFile | title, artist, duration, fileName, fileFormat, fileSize, folderPath |

## 技术信息
- SwiftUI + AVFoundation/AVPlayer
- iOS 18.2+
- 只支持本地音频文件：mp3, wav, m4a, aac
